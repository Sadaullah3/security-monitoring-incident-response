Incident Scenarios & Response Steps
Incident Response Playbook
Incident 1: Brute Force Attack
Detection

Alert from login monitoring rule.

Classification

Severity: High

Response Steps

Block attacking IP temporarily

Reset targeted user passwords

Enable CAPTCHA

Investigate IP reputation

Review logs for successful login

Containment

Rate limit login endpoint.

Incident 2: Admin Account Compromise
Detection

Admin login from unusual location.

Classification

Critical

Response

Immediately disable admin account

Force password reset

Check for configuration changes

Review audit logs

Enable MFA if not active

Incident 3: Suspicious File Upload
Detection

Malicious file type detected.

Response

Remove file

Scan system for compromise

Block dangerous extensions

Add server-side validation
